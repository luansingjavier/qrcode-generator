# QR Code Generator with Logo

A beautiful, modern web application for generating QR codes with custom logo support.

## Features

- ✨ Generate QR codes from URLs or any text
- 🖼️ Add custom logos to QR codes
- 🎨 Adjustable logo size (10-30% of QR code)
- 📏 Customizable QR code size (200-600px)
- 🔧 Error correction levels (L, M, Q, H)
- 💾 Download QR codes as PNG images
- 📱 Fully responsive design

## Usage

1. Open `index.html` in a web browser
2. Enter a URL or text in the input field
3. (Optional) Upload a logo image
4. Adjust settings as needed
5. Click "Generate QR Code"
6. Download your QR code

## Technologies

- HTML5
- CSS3 (Modern gradients and animations)
- JavaScript (Vanilla JS)
- QRCode.js library (CDN)

## Browser Support

Works in all modern browsers that support:
- Canvas API
- FileReader API
- ES6+ JavaScript
